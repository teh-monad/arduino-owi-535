Arduino example for work with OWI-535 arm through bluetooth.

## Details

* Arduino UNO v3 or etc.;
* 5x L9110H drivers;
* HC-06 bluetooth;
* breadboard/arduino motor shield;

## Usage

You should write personal function to any action or use exists.
